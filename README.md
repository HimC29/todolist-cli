# todolist-cli

![todolist CLI](https://img.shields.io/badge/todolist-CLI-lightgrey?labelColor=blue)

A lightweight **CLI program** to manage tasks using a text file in the current directory.

> ⚠️ **Early Release / In Development**  
> todolist-cli is still in active development. Some features may be missing or incomplete.

---

## Releases

- **Debian** – [https://github.com/HimC29/todolist-cli-deb](https://github.com/HimC29/todolist-cli-deb)    

## Features

- **Add Tasks** – Quickly add new tasks from the terminal.  
- **View Tasks** – Display all current tasks stored in `tasks.txt`.  
- **Reload Tasks** – Reload the task list from the file without restarting the program.  
- **Clear Screen** – Clean up your terminal view for easier reading.  
- **Simple CLI Navigation** – Intuitive commands:  
  - `w` → Add new task  
  - `r` → Reload tasks  
  - `c` → Clear screen  
  - `q` → Quit program
